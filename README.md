# Number-Guessing-Game 
https://roadmap.sh/projects/number-guessing-game

Game prompts the player to select a difficulty. If an incorrect difficulty is selected the user will have to select again

Three different modes Easy (10 tries), Medium (5 tries), Hard (3 tries). 

Player will then be prompted to make a guess and will be told how many guesses remain if the correct number is not guessed.

When the player makes the correct guess or runs out of guesses, the player will be prompted if they wish to play again.
